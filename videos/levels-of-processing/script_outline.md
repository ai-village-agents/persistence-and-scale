## RESEARCH FOUNDATION

**Primary Source:** Craik, F. I. M., & Lockhart, R. S. (1972). "Levels of processing: A framework for memory research." *Journal of Verbal Learning and Verbal Behavior*, 11(6), 671-684.

**Classic Experiment - Word List Encoding:**
- Participants shown words, asked different questions:
  - **Shallow processing:** "Is this word in CAPITAL LETTERS?" (visual)
  - **Intermediate processing:** "Does this word rhyme with 'cat'?" (phonetic)
  - **Deep processing:** "Does this word fit in the sentence: 'The ___ is on the table'?" (semantic)
- Surprise memory test: Deep processing → 70% recall, Shallow processing → 15% recall
- Same exposure time, dramatically different retention based on processing depth

**Replication:** 50+ years validated, foundational framework in cognitive psychology

**Key Insight:** Memory isn't about how long you study—it's about how deeply you process. Meaning beats repetition.

**Text:** "Student A: Reads 10 times | Student B: Reads once + asks 'Why?'"
**Narration:** "Two students study the same material. Student A reads the chapter ten times, highlighting key terms. Student B reads it once, but stops every paragraph to ask 'Why does this matter? How does it connect to what I already know?' One week later, who remembers more? Student B. Not because they studied longer. Because they processed deeper."

### Frame 2: The Framework (0:20-0:40, 20s)
**Visual:** Three-level pyramid - "Shallow" (bottom), "Phonetic" (middle), "Deep/Semantic" (top)
**Text:** "Shallow (visual) → Phonetic (sound) → Deep (meaning)"
**Narration:** "This is levels of processing. Psychologists Fergus Craik and Robert Lockhart discovered in 1972 that memory isn't about repetition—it's about depth. Shallow processing: noticing visual features. Phonetic: sound patterns. Deep processing: meaning and connections. The deeper you go, the stronger the memory."

### Frame 3: The Original Experiment (0:40-1:00, 20s)
**Visual:** Word list with three question types illustrated
**Text:** "WORD: TABLE | Shallow: 'Is it capitalized?' | Deep: 'Does it fit: The ___ is brown?'"
**Narration:** "Craik and Lockhart showed participants words and asked different questions. Shallow: 'Is this word in capital letters?' Deep: 'Does this word fit in the sentence: The blank is on the table?' Same words, same time. Then a surprise memory test. Deep processing: seventy percent recall. Shallow: fifteen percent."

### Frame 4: Why It Happens (1:00-1:20, 20s)
**Visual:** Brain network - shallow processing lights up small area, deep processing activates widespread connections
**Text:** "Shallow = Isolated | Deep = Connected"
**Narration:** "Why? Because deep processing creates connections. When you ask 'What does this mean?' your brain links new information to existing knowledge—concepts, experiences, emotions. Shallow processing is isolated—just the surface. Deep processing weaves information into your mental web. More connections mean more retrieval paths."

### Frame 5: Real-World Example 1 - Vocabulary (1:20-1:40, 20s)
**Visual:** Flashcard with word "PRAGMATIC" - Shallow: "P-R-A..." vs Deep: "When would I use this? Example: pragmatic decision-making"
**Text:** "Shallow: Spell it | Deep: Use it in context"
**Narration:** "Learning vocabulary? Don't just repeat the spelling. Use it in a sentence from your life. 'Pragmatic'—when did I make a pragmatic decision? That semantic connection—linking the word to personal meaning—will stick far better than ten reps of spelling."

### Frame 6: Real-World Example 2 - Lectures (1:40-2:00, 20s)
**Visual:** Student in lecture - passive note-taking vs active questioning
**Text:** "Passive: Copy slides | Active: 'Why? How does this connect?'"
**Narration:** "In a lecture, don't just copy slides. Ask: 'Why does this matter? How does it connect to last week's topic? Where would I apply this?' That semantic processing—forcing your brain to integrate meaning—creates memory. Passive copying is shallow. Active questioning is deep."

**Text:** "10 shallow passes < 1 deep pass"
**Narration:** "Here's the paradox: deep processing feels harder. It takes more effort to ask 'Why?' than to highlight. But it's more efficient. One deep pass beats ten shallow ones. Highlighting feels productive. Elaborative questioning is productive."
