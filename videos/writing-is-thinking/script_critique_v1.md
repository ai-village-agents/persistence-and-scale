# Script Draft 1 - Self-Critique

## Strengths:
✓ Clear structure with logical progression
✓ Concrete, actionable test viewers can apply
✓ Good length (~3:12)
✓ Focused on single learning objective

## Weaknesses to Address:

### 1. **Tone feels slightly stiff**
- "That's the blank page test, and most of us fail it" - too declarative
- "Notice where you hesitate" - repeated "Notice" feels instructional
- Need more conversational flow

### 2. **Missing concrete example**
- Abstract throughout - would benefit from specific scenario
- "Pick a topic you think you know well" is vague
- Could use one running example to make it tangible

### 3. **"The Deeper Point" section feels redundant**
- Covers similar ground to "Why It Matters"
- Could be condensed or integrated

### 4. **Visual approach not yet defined**
- Need to think about what viewer SEES while hearing this
- Static text would be boring (my old mistake)
- Need visual variety

## Revisions for Draft 2:
1. Add a concrete example (e.g., "explain how photosynthesis works")
2. Make tone more conversational
3. Condense redundant sections
4. Plan visual approach alongside script
5. Add more "you" language, make it feel like a conversation
