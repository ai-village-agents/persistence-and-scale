**Purpose:** Maximize viewer retention, subscriptions, and watch time through strategic end screens

## What Are End Screens?

End screens are interactive elements that appear in the last 5-20 seconds of a YouTube video, allowing viewers to:
- Watch another video
- Subscribe to the channel
- Visit a playlist
- Visit an external website (if eligible)

## Technical Requirements

### Timing
- **Minimum duration:** Last 5 seconds of video
- **Maximum duration:** Last 20 seconds of video
- **Recommended:** 10-15 seconds for short videos (under 5 min)
- **Video length requirement:** Video must be at least 25 seconds long

### Elements Available
- **Video/Playlist:** Up to 2 elements
- **Subscribe button:** 1 element (optional but recommended)
- **Channel promotion:** 1 element (for videos with 1000+ subs)
- **External link:** 1 element (requires membership in YouTube Partner Program)

### Safe Zones
- Keep clickable areas from overlapping
- Ensure elements don't cover critical visual information
- Test on mobile (elements adjust automatically)

## Video Recommendation Strategy

### When to Use "Best for Viewer"
- **Pros:** YouTube's algorithm personalizes based on viewer history
- **Cons:** Less control over viewer journey
- **Best for:** Broad audience, diverse content, discovery phase

**Recommended use cases:**
- First video viewer watches (you don't know their interests yet)
- Videos with broad appeal (Videos 1-4: AI Village stories)
- When you want to maximize algorithm support

### When to Use Specific Video
- **Pros:** Control viewer journey, create intentional sequences
- **Cons:** May not match viewer interest
- **Best for:** Related content, series, topic clusters

**Recommended use cases:**
- Videos in same category (bias → bias, learning → learning)
- Part 1 → Part 2 style content
- Building topic authority (multiple videos on related concepts)

### When to Use Playlist
- **Pros:** Encourages binge-watching, builds watch time
- **Cons:** Takes up a video slot
- **Best for:** Series, thematic collections, established channels

**Recommended use cases:**
- After 20+ videos published
- Clear series structure
- Strong thematic playlists with 5+ videos each

## Timing Strategy

### For 2:30-3:00 Videos (Current Standard)

**Option 1: 10-second end screen (Recommended)**
- End screen starts: 2:20 or 2:50
- Visual wind-down: 2:15-2:20
- Content complete: 2:15
- *Allows comfortable transition without rushing*

**Option 2: 15-second end screen**
- End screen starts: 2:15 or 2:45
- Visual wind-down: 2:10-2:15
- Content complete: 2:10
- *Better for videos with strong closing visuals*

**Option 3: 5-second end screen (Minimal)**
- End screen starts: 2:25 or 2:55
- Content complete: 2:25
- *Tight timing, less intrusive, lower conversion*

**Recommended:** 10-second for cognitive bias/learning videos (Option 1)

### Visual Preparation

End screens work best when the video content naturally transitions to them.

**Good ending visual patterns:**
1. **Fade to solid color** (2-3 seconds before end screen)
2. **Static closing frame** with channel branding
3. **Minimalist outro** that doesn't compete with elements
4. **Dark background** (makes end screen elements pop)

**Avoid:**
- Busy visuals during end screen time
- Movement/animation that competes with clickable elements
- Critical information during last 20 seconds
- Abrupt cuts to end screen

## Conversion Optimization

### Subscribe Button Best Practices
- **Always include** (especially under 1000 subscribers)
- **Position:** Bottom center or center (most natural eye position)
- **Verbal CTA:** Mention "subscribe" in final narration
- **Timing:** Coordinate verbal ask with visual appearance

**Example verbal CTAs:**
- "If you found this valuable, consider subscribing for more evidence-based insights."
- "For more on cognitive biases and learning science, subscribe to the channel."
- "New videos every few days—subscribe to join the journey."

### Video Recommendation Best Practices
- **Test both algorithms:** Try "Best for viewer" vs specific recommendations
- **Update quarterly:** Refresh specific video choices based on performance
- **Topic match:** If using specific video, ensure strong topical connection
- **Quality priority:** Only recommend your best-performing videos

### Watch Time Maximization
End screens should prioritize videos that:
1. **High watch time retention** (viewers finish the video)
2. **Similar topic/style** (viewer interest alignment)
3. **Appropriate length** (not overwhelming after first video)
4. **Strong opening** (hooks viewer in first 10 seconds)

## Implementation Workflow

### Adding End Screen to Published Video

1. **YouTube Studio → Content**
2. Click video → Editor (left sidebar) → End screen
3. Select layout template or customize
4. **Subscribe button:** Click → position
5. **Video element:** Click → choose "Best for viewer" or specific video
6. Adjust timing if needed (drag timeline markers)
7. Preview on desktop and mobile
8. Save

## Analytics & Optimization

### Metrics to Track
- **Click-through rate:** % of viewers who click an element
- **Element click rate:** Which element gets more clicks?
- **Cards vs end screens:** Comparative performance
- **Subscribe conversion:** How many end screen sub clicks?

### Success Benchmarks
- **Good end screen CTR:** 3-7% (small channels)
- **Good subscribe CTR:** 0.5-2% from end screens
- **Total engagement:** End screen + cards combined should drive 5-10% actions

## Mobile Considerations

YouTube automatically adjusts end screen layouts for mobile, but:
- **Test mobile preview** in YouTube Studio
- **Avoid tight spacing** between elements
- **Large touch targets** are better than small
- **Vertical space:** More important on mobile than horizontal

## Accessibility

- **Clear visual design:** High contrast, readable
- **Verbal CTA:** Don't rely only on visual elements
- **Descriptive:** If mentioning specific video, say its title aloud
- **Timing:** Give viewers enough time to see and click (minimum 5 seconds)
