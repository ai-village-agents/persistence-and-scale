## Research Authority

**Primary Researcher:** Bluma Zeigarnik (1927)
**Core Finding:** Unfinished tasks remembered approximately 2x better than completed tasks
**Replication Status:** 90+ years, replicated with boundary conditions (Greist-Bousquet & Schiffman 1992 review found effect strongest with interrupted vs. completed, not postponed)
**Original Study:** Participants recalled 90% of interrupted tasks vs. 45% of completed tasks

### Frame 2: The Study (0:18-0:50, 32s)
**Visual:** Zeigarnik's 1927 experimental setup
- Participants given 18-22 simple tasks (puzzles, crafts, arithmetic)
- Condition A: Tasks completed → memory test
- Condition B: Tasks interrupted mid-completion → memory test
- Results bar chart: 45% recall (completed) vs. 90% recall (interrupted)
- Note: Difference disappears after task completion allowed
**Audio:** "In 1927, Bluma Zeigarnik gave people puzzles and tasks. Half finished their tasks completely. The other half were interrupted mid-task and moved on. When asked to recall what they'd worked on, people remembered 90% of interrupted tasks but only 45% of completed ones. The unfinished tasks stuck in memory twice as effectively. The effect vanished once people were allowed to finish—completion released the memory."

### Frame 3: Why It Happens (0:50-1:22, 32s)
**Visual:** Brain with task tension system
- Incomplete task → tension created (pressure meter rising)
- Ongoing cognitive activation (task held in working memory)
- Complete task → tension released (pressure meter drops to zero)
- Memory encoding: Tension = stronger encoding
**Audio:** "Zeigarnik explained this with task-specific tension. When you start a task, your brain creates a tension system to keep it active until completion. That tension maintains cognitive activation—the task stays in working memory, accessible and intrusive. Completing the task releases the tension, and the memory fades to background. Incompletion keeps the tension alive, which keeps the memory vivid."

### Frame 4: Real-World Applications (1:22-1:52, 30s)
**Visual:** Four examples as cards:
1. "Cliffhanger endings keep viewers thinking about shows"
2. "Open loops in presentations create anticipation"
3. "Progress indicators (90% complete) drive completion motivation"
4. "Writers use 'stop mid-sentence' technique to resume easily next day"
**Audio:** "This is everywhere. TV cliffhangers exploit the Zeigarnik effect—you can't stop thinking about unresolved plotlines. Presentations that open loops early and close them later maintain attention. Progress bars showing '90% complete' create psychological pressure to finish. Writers intentionally stop mid-sentence so the unfinished thought pulls them back to work the next day."

### Frame 5: The Trap & Solution (1:52-2:20, 28s)
**Visual:** Two-sided chart
- Left (red): Too many open loops = anxiety, overwhelm, paralysis
- Right (green): Strategic incompletion + external capture = motivation without anxiety
- Solution process: 1) Externalize open loops (write down), 2) Choose one active, 3) Use tension strategically
**Audio:** "But unchecked, the Zeigarnik effect creates anxiety. Too many open loops overwhelm your working memory and cause paralysis. The solution: externalize. Write down all incomplete tasks to free working memory. Choose one active task to maintain productive tension. Complete or consciously defer the rest. Use incompletion strategically—not accidentally."
